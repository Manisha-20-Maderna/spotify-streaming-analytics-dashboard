# 🎵 Spotify Streaming Analytics Dashboard

An interactive **Power BI dashboard** built to analyze Spotify streaming data, providing insights into artist performance, track popularity, playlist impact, release trends, and streaming behavior through interactive visualizations and KPIs.

---

## 📌 Project Overview

This project explores Spotify's streaming dataset using **Power BI**, **Power Query**, and **DAX**. The dashboard helps identify streaming trends, top-performing artists and tracks, the relationship between popularity and streams, and the impact of playlists on song performance.

## 🎯 Objectives

- Analyze Spotify streaming performance.
- Identify the top-performing artists and tracks.
- Compare Spotify streams with YouTube views.
- Understand the impact of playlist placements.
- Analyze streaming trends by release year and release day.
- Generate business insights using interactive visualizations.

---

## 📊 Dashboard Features

- 📌 KPI Cards
  - Total Artists
  - Total Tracks
  - Total Spotify Streams
  - Total YouTube Views
  - Average Popularity

- 📈 Interactive Visualizations
  - Top 10 Tracks by Spotify Streams
  - Top Artists by Spotify Streams
  - Spotify Popularity vs Spotify Streams
  - Playlist Count vs Spotify Streams
  - Spotify Streams vs YouTube Views
  - Average Spotify Streams by Release Year
  - Spotify Streams by Release Day
  - Average Spotify Streams by Explicit Content

- 🎛 Interactive Slicers
  - Artist
  - Release Year
  - Explicit Track

- 💡 Business Insights

---

## 📈 Key Insights

- Top tracks contribute the largest share of Spotify streams.
- Higher Spotify popularity is associated with higher stream counts.
- Songs featured in more playlists generally receive more streams.
- Friday has the highest number of high-streaming song releases.
- A small number of top artists generate a large share of total streams.
